# FraternalFinder.com

**The most comprehensive free directory of fraternal lodges in the United States.**

Live Site: [https://fraternalfinder.com](https://fraternalfinder.com)

FraternalFinder.com is a free, open-source directory featuring **5,485+ fraternal lodge locations** across all 50 US states. Whether you are looking to join a lodge, visit while traveling, or learn about fraternal organizations in your area, FraternalFinder makes it easy to search by state, city, or organization.

## Features

- **8,771 indexed pages** covering every lodge with address, phone, website, and map
- **State-level directories** for all 50 states with city-level drill-down
- **Organization filtering** - browse by Elks, Moose, Eagles, Knights of Columbus, Lions, or Odd Fellows
- **SEO-optimized** static site with FAQPage schema, BreadcrumbList, and LocalBusiness markup
- **Mobile-responsive** design that works on any device
- **Fast loading** - static HTML served via Cloudflare Pages CDN
- **No login required** - all data is freely accessible

## Tech Stack

- **Static Site Generator:** Custom Python (~5,200 lines) generating 8,771 HTML pages
- **Hosting:** Cloudflare Pages (free tier)
- **CDN/SSL:** Cloudflare (automatic HTTPS, global edge caching)
- **Analytics:** Google Analytics 4
- **Structured Data:** JSON-LD (FAQPage, BreadcrumbList, LocalBusiness, WebSite)
- **Search Indexing:** IndexNow API for instant crawl notifications
- **Email:** Cloudflare Email Routing (contact@fraternalfinder.com)

## About Fraternal Organizations

Fraternal lodges have been a cornerstone of American civic life since the early 1800s. These member-driven organizations collectively contribute **billions of dollars annually** in charitable giving, including:

- **$4.5 million+** in Elks scholarships each year
- **$2 billion+** in Knights of Columbus charitable giving (2023)
- **$135 million+** raised annually by Eagles for community causes
- **$100 million+** invested in Moose youth programs through Mooseheart and Moosehaven

FraternalFinder.com was created to help people discover and connect with these vital community organizations.
